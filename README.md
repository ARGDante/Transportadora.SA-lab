# Transportadora.SA-lab
Este repositorio contiene documentación detallada de máquinas de Capture The Flag (CTF) y
entornos de práctica que he resuelto. Cada informe representa un análisis completo de seguridad,
explotación de vulnerabilidades y propuestas de mitigación.

## Objetivo
Documentar mi proceso de aprendizaje en ciberseguridad

Demostrar habilidades técnicas en pentesting

Crear un recurso educativo para otros entusiastas de la seguridad

Mantener un registro organizado de mis progresos
